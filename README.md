SourKnight 项目说明文档
同济大学软件学院2020年面向对象程序设计课程大项目


游戏玩法

2D角色扮演游戏，玩家操控主角深入地牢消灭怪物，获得各种武器，最终击败boss。

打开压缩包，运行setup.exe文件，安装游戏

进入游戏后，关闭输入法，点击开始游戏，新游戏，进入游戏界面。点击灰色人物进行英雄选择，随后通过右边房间内传送门进入关卡。

操作	效果
按下`WSAD`键	控制英雄上、下、左、右移动
点击鼠标左键	英雄向鼠标点击处攻击
按下`Q`键	拾取道具 /祭拜雕像
按下`R`键	拾取并切换武器
按下`E`键	释放技能
消灭关卡内所有怪物进入下一关。
